# shitty-game-of-life
shitty Game of Life — потому что жизнь дерьмова, и симуляция дерьмова тоже.

Клавиши:
Space — следующий шаг (с латентами, чтобы успели заскучать)
N — следующий шаг
C — новое поле (сеянное текущим временем в миллисекундах, потому что rand() для слабаков)
X — очистить поле (аналог кнопки «начать сначала», но без оптимизма)
Esc — выход (не там написал функцию выхода(, но мне лень переделывать )
